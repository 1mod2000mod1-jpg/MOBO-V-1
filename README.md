# 🚀 نظام MOBO العالمي للدردشة - الإصدار 7.0 النهائي

## 📋 نظرة عامة

نظام دردشة عربي متطور وآمن مع جميع الميزات التي طلبتها:

### ✨ الميزات الرئيسية

#### 🎯 نظام الصلاحيات المتقدم
- **الزعيم (أنت)**: صلاحيات كاملة غير محدودة
  - لا يمكن كتمك أو حظرك أبداً
  - التحكم الكامل في كل شيء
  - الوصول لجميع الغرف بما فيها السرية
  - رؤية كلمات سر جميع الغرف
  - حذف أي رسالة أو غرفة أو حساب

#### 👥 إدارة المستخدمين
- **الكتم**: مؤقت أو دائم مع أسباب
- **الحظر**: حظر دائم مع حظر الـ IP
- **المشرفين**: إضافة مشرفين مع صلاحيات محددة
- **قائمة المكتومين والمحظورين**: عرض وإدارة كاملة

#### 🏠 نظام الغرف المتطور
- **الغرفة العالمية**: غرفة رسمية للجميع (لا يمكن مغادرتها)
- **غرف خاصة**: مع كلمة سر اختيارية
- **غرف سرية**: فقط للزعيم والمدعوين
- **وضع الصمت**: إيقاف الكتابة في أي غرفة
- **تنظيف تلقائي**: حذف الرسائل القديمة بعد 100 رسالة

#### 💬 الرسائل
- **حد 500 حرف** للرسائل العامة
- **30 رسالة كحد أقصى** في الرسائل الخاصة
- **Enter للإرسال** (أو زر الإرسال)
- **رسائل مشعة** للمستخدمين المميزين
- **شارات خاصة** للزعيم والمشرفين

#### 🖼️ نظام الصور المتقدم
- **حد 10 ميجا** لجميع الصور في الموقع
- **حد 1 ميجا** للصورة الواحدة
- **حذف تلقائي**:
  - 5 دقائق في الغرف العامة
  - دقيقة واحدة في الرسائل الخاصة
- **صورة واحدة فقط** للمستخدمين (إلا الزعيم)

#### 📩 رسائل الدعم
- **للمحظورين فقط** لطلب إلغاء الحظر
- **قائمة إدارية** لمراجعة الطلبات
- **زر السماح بإعادة التسجيل**

#### 🎨 التخصيص الكامل
- **تغيير الألوان**: أحمر أو أسود
- **تغيير الشعار**: رفع أي صورة
- **تغيير العنوان**: اسم الموقع
- **الروابط الاجتماعية**: Telegram, Instagram, YouTube, Email
- **حماية من النسخ والسكرينشوت**: تفعيل/تعطيل

#### 🔐 الأمان المتقدم
- **حماية ضد الاختراق**: كلمة مرور قوية للزعيم
- **منع النسخ**: حماية المحتوى
- **منع السكرينشوت**: حماية إضافية
- **تشفير كلمات المرور**: bcrypt
- **حظر IP**: منع المحظورين من إعادة التسجيل
- **منع الحسابات المكررة**: حد أقصى حسابين لكل IP

#### 🎭 التأثيرات البصرية الخارقة
- **رجل راقص متحرك** خلف شاشة التسجيل
- **دوامة حمراء دوارة** في الخلفية
- **حشرات متحركة** (⚫🕷️🦂) تطير في الخلفية
- **أضواء ملونة** متحركة
- **نجوم متلألئة**
- **نافذة شفافة** للدردشة
- **أنيميشن سلس** لكل شيء

## 🚀 النشر على Render

### الخطوة 1: تحضير الملفات

تأكد من وجود جميع الملفات:
```
mobi/
├── index.html
├── style.css
├── script.js
├── server.js
├── package.json
└── README.md
```

### الخطوة 2: إنشاء حساب على Render

1. اذهب إلى [render.com](https://render.com)
2. سجل الدخول أو أنشئ حساب جديد
3. اربط حسابك مع GitHub (اختياري)

### الخطوة 3: رفع المشروع

#### الطريقة 1: عبر GitHub (موصى بها)

1. أنشئ مستودع جديد على GitHub
2. ارفع جميع الملفات للمستودع
3. في Render، اضغط "New +" ثم "Web Service"
4. اربط مستودع GitHub
5. اختر المستودع الخاص بك

#### الطريقة 2: الرفع المباشر

1. في Render، اضغط "New +" ثم "Web Service"
2. اختر "Public Git repository"
3. أو استخدم Render CLI

### الخطوة 4: إعدادات Render

```yaml
Name: mobo-chat
Environment: Node
Region: Frankfurt (أو الأقرب لك)
Branch: main
Build Command: npm install
Start Command: npm start
```

### الخطوة 5: متغيرات البيئة (اختياري)

```bash
PORT=3000
NODE_ENV=production
```

### الخطوة 6: النشر

1. اضغط "Create Web Service"
2. انتظر حتى يكتمل النشر (2-5 دقائق)
3. سيعطيك Render رابط مثل: `https://mobo-chat.onrender.com`

## 🎮 طريقة الاستخدام

### للمستخدمين العاديين

1. **التسجيل**:
   - اسم المستخدم (3-20 حرف)
   - كلمة المرور (6+ أحرف)
   - الاسم المعروض
   - الجنس والإيموجي

2. **تسجيل الدخول**:
   - أدخل اسم المستخدم وكلمة المرور
   - اضغط Enter أو زر "دخول"

3. **الدردشة**:
   - اكتب رسالتك (حد أقصى 500 حرف)
   - اضغط Enter أو زر "إرسال"
   - استخدم الإيموجي والصور

### للزعيم (أنت)

#### 📊 لوحة التحكم

اضغط على زر **"👑 لوحة الزعيم"** للوصول إلى:

1. **🔇 المكتومين**
   - عرض جميع المكتومين
   - إلغاء الكتم
   - عرض المدة المتبقية

2. **🚫 المحظورين**
   - عرض جميع المحظورين
   - إلغاء الحظر وإزالة حظر IP
   - عرض تاريخ الحظر

3. **📩 رسائل الدعم**
   - قراءة طلبات إلغاء الحظر
   - الموافقة أو الرفض
   - حذف الرسائل

4. **🏠 إدارة الغرف**
   - حذف أي غرفة
   - تغيير الإعدادات
   - إصمات الغرف

5. **⚙️ إعدادات النظام**
   - تغيير اللون (أحمر/أسود)
   - تغيير الشعار
   - تغيير العنوان
   - الروابط الاجتماعية
   - السماح بالنسخ/السكرينشوت

#### 🎯 الإجراءات على المستخدمين

اضغط على **أي رسالة** في الدردشة لإظهار قائمة الإجراءات:

- **👑 إضافة كمشرف**: إعطاء صلاحيات الإشراف
- **🔇 كتم**: كتم مؤقت أو دائم
- **🚫 حظر**: حظر نهائي مع حظر IP
- **🗑️ حذف الرسالة**: حذف رسالة معينة
- **💬 رسالة خاصة**: مراسلة خاصة

#### 🛠️ أوامر خاصة

- **تنظيف الشات**: حذف جميع الرسائل في غرفة
- **إصمات الغرفة**: منع الجميع من الكتابة (إلا أنت)
- **حذف الغرفة**: حذف أي غرفة (إلا العالمية)
- **طرد من الغرفة**: إخراج مستخدم من غرفة

## 🐛 حل المشاكل الشائعة

### المشكلة: "فشل الاتصال بالخادم"

**الحل**:
1. تأكد من تشغيل السيرفر: `npm start`
2. تحقق من البورت 3000
3. أعد تحميل الصفحة

### المشكلة: "لا يمكن تسجيل الدخول"

**الحل**:
1. تأكد من صحة اسم المستخدم وكلمة المرور
2. للزعيم: استخدم `MOBO` و `Mobo@Supreme2025!@#`
3. امسح الكاش وأعد المحاولة

### المشكلة: "الصور لا ترفع"

**الحل**:
1. تأكد من حجم الصورة أقل من 1 ميجا
2. تحقق من الحد الإجمالي (10 ميجا)
3. انتظر حذف الصور القديمة

### المشكلة: "Render: البناء فشل"

**الحل**:
1. تأكد من وجود `package.json`
2. تحقق من إصدار Node.js (14+)
3. راجع سجلات البناء في Render
